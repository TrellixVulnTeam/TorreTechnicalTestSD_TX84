Torre Technical Test: The idea of the demo is to improve the way in which the users find relevant data when they are looking for information.

What I saw, is that the search engine, brings a lot of information to the client side, so it takes time to retrieve this information, situation that is increased when they want to add a filter and repeat the process.

The idea is to add a filter over the loaded data, so the user can filter this information more quickly over the data loaded.

In the example we retrieve information using the Post Method: https://search.torre.co/people/_search, and we work with the results of the Search filtering the information with a SearchBar.
